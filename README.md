Solana MEV bot & degen lab. Black and yellow, loud and honest.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

1. Push to GitHub
2. Import to Vercel
3. Deploy

## Content Editing

All copy lives in `config/site-config.ts` - edit there to change text, links, etc.
